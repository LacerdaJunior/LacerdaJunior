## 🏆 Projeto em Destaque

**[Kamalla: Educação Gamificada](https://liberal.com.br/cidades/americana/etec-de-nova-odessa-vence-mostra-de-ciencia-e-tecnologia-do-instituto-3m?fbclid=PAZXh0bgNhZW0CMTEAc3J0YwZhcHBfaWQMMjU2MjgxMDQwNTU4AAGnvJbJ8OTVwhP0nlBDEUcDX_mUO5pV24CxqDw0lIpm3jl06y6umFvtCKKvGi8_aem_M76r3KPq0GrPPEsrRkXVDg)** 🥇 **1º Lugar em Ciências Humanas** na *13ª Mostra de Ciência e Tecnologia Instituto 3M* (2025).

Plataforma de metodologia ativa focada em recuperar o engajamento e a atenção dos estudantes. Desenvolvida em equipe durante a Etec de Nova Odessa, a plataforma permite que docentes criem turmas, cadastrem alunos e disponibilizem atividades personalizadas utilizando elementos de gamificação escolar. O projeto também foi credenciado para a Febrace (USP).
